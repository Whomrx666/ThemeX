# ThemeX
![ThemeX preview](ThemeX.jpg)

## introduction
ThemeX is a tools created to make your terminal look beautiful and pretty and there are lots of different themes in it that you can choose from.

## Instalations
```
$ pkg update -y && pkg upgrade -y
$ pkg install git -y
$ git clone https://github.com/Whomrx666/ThemeX.git 
$ cd ThemeX
$ bash install.sh

```
## Run
```
$ bash ThemeX.sh
```

## Instructions
- **First**: Install tools according to the instructions above and don't forget to activate Tor first.
- **Second**: Run the tool and select one of the banner lists in the menu.
- **Third**: After selecting the tool, it will process the theme that will be applied.
- **Last**: Wait a few seconds and boom your terminal will adjust to the theme you chose.
